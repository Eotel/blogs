## 概要

**キュプラ（cupro）** は、**銅アンモニア溶液**（テトラアミンジアクア銅(II) 水酸化物 [Cu(NH₃)₄(H₂O)₂](OH)₂、別名 Schweizer's reagent / Cuoxam / cuprammonium hydroxide）に高純度セルロースを溶解し、紡糸口金から凝固浴に押し出して再生する **再生セルロース繊維**。広義には「レーヨン」の一種（英語圏では *cuprammonium rayon* と呼ぶ）だが、日本の家庭用品品質表示法・JIS L 0204-2 では「レーヨン」（ビスコース法）と別の指定用語として運用される。

**狭義のレーヨン（ビスコース法）との違いの本質は原料ではなく製法と繊維設計** にある。「コットンリンター由来だから高級」という売り場の説明は、原料の物語に技術的本質をすり替えたマーケティング。

## レーヨンファミリーにおける位置

「レーヨン」という語は広義と狭義で意味が異なる。

| 表記 | 意味 | 製法 |
|---|---|---|
| 広義のレーヨン | 再生セルロース繊維全般 | ビスコース・銅アンモニア・モダール・リヨセル すべて含む |
| 狭義のレーヨン | ビスコース法による繊維 | アルカリ処理 → CS₂ で誘導体化 → 酸浴で再生 |
| キュプラ | 銅アンモニア法による繊維 | Schweizer's reagent で溶解 → 凝固浴で再生 |

ISO 2076（現行 2021 年版）はファミリー単位で `viscose` / `modal` / `lyocell` / `cupro` を別の generic name として列挙する。JIS L 0204-2:2020 も「レーヨン (2001) — ビスコース法」「キュプラ (2005) — 銅アンモニア法」を別の用語として定義している。

## 銅アンモニア法 vs ビスコース法

| 観点 | ビスコース法（狭義レーヨン）| 銅アンモニア法（キュプラ）|
|---|---|---|
| 溶剤 | NaOH + CS₂（キサントゲン酸誘導体経由）| 銅アンモニア溶液 [Cu(NH₃)₄(H₂O)₂](OH)₂（Schweizer's reagent）|
| 凝固浴 | 硫酸を含む酸浴 | 水・希酸 |
| 繊維形態 | 短繊維（ステープル）が多い | 長繊維（フィラメント）に向く |
| 断面 | 不規則（菊花状）| 比較的丸く滑らか |
| フィラメント細さ | 中〜太め | 細く設計しやすい |
| 風合い | 一般用途・廉価帯 | 滑り・光沢・吸放湿性に優れる |
| 環境課題 | CS₂ の取り扱い | 銅・アンモニア廃液の管理（クローズドループ回収で対処）|
| 主用途 | 衣料一般・不織布・タイヤコード | スーツ高級裏地・スカーフ・インナー |

## 原料 — コットンリンターは「都合のよい高純度セルロース源」

キュプラの原料は **コットンリンター**（綿実の周りに残る短繊維、綿実油製造の前駆体として回収される副産物）。コットンリンターを採用する技術的理由は次の通り。

- 高純度セルロースが得やすい
- リグニン・ヘミセルロースが少ない
- 銅アンモニア溶液への安定した溶解性
- 紡糸口金を細く保つ均一な重合度分布

ただし **コットンリンターはキュプラを定義する本質ではない**。原理的には木材パルプ（高純度の dissolving pulp に限る）からもキュプラを製造できる（戦後日本でも研究実績あり）。**定義は「銅アンモニア法で再生した繊維」であって、原料ではない**。

## ベンベルグ — 世界唯一のキュプラ生産者

現代のキュプラ生産は **旭化成のベンベルグ™（Bemberg）に実質的に統一されている**。

- 1928 年: 旭化成が J. P. Bemberg（ドイツ）と技術導入の正式契約に調印
- 1931 年: 国内で Bemberg™ キュプラ生産開始
- 20 世紀末: ドイツ側の生産が縮小・終了
- 2000 年代: 他社の cupro 生産も停止
- **2009 年: 旭化成が世界で唯一の cupro 生産者となる**

旭化成は自社サイトで "*one-of-a-kind producer of cupro in the world*" と明記しており、Bemberg™ は商標として旭化成が保有・運用している。「キュプラ＝ベンベルグ」という市場認識には、**世界で 1 社だけが量産している希少素材** という現実が裏付けられている。

## 制度上の位置（標準・規格）

| 制度 | 扱い |
|---|---|
| **ISO 2076:2021** | `cupro` を独立の generic name として列挙 |
| **JIS L 0204-2:2020** | 「キュプラ」（2005 番、銅アンモニア法によるセルロース繊維）として独立用語化 |
| **消費者庁・家庭用品品質表示法** | 「レーヨン」とは別の指定用語「キュプラ」を運用 |
| **16 CFR §303.7（米国 FTC）** | cupro は FTC が従来から承認している generic fiber name で、単独項目では未列挙だが、§303.7 が **ISO 2076:2013(E) を incorporate by reference** することで ISO 標準と整合化されている |

## 識別チートシート

実物の素材を見るときの目安。

1. 表示に「**キュプラ**」「**cupro**」「**Bemberg**」「**ベンベルグ**」のいずれかがあればキュプラ
2. 「**レーヨン**」「**rayon**」「**viscose**」「**ビスコース**」表記はキュプラではない（狭義のレーヨン）
3. スーツ・コートの裏地で、**シルクのような滑りと光沢、軽い静電気の起きにくさ** を感じればキュプラの可能性が高い
4. 廉価ジャケットの安い裏地（張り付き感・静電気・ザラつき）はビスコースレーヨンの可能性が高い
5. **「コットンリンター由来だから高級」という売り文句は半分マーケティング**。本質は銅アンモニア法と細い長繊維設計にある
